Netflix movies data analysis

- Datasets used : available in the Github folder
- Program used : Python
- Libraries used : Pandas, Numpy, datetime

The project is an exploratory data analysis about Netflix movies. The Movies in the datasets were explored and analyzed to further understand them. The program
used to manipulate the data was Python, with the help of libraries like Pandas,and Numpy. The Python libraries used helped with data
wrangling tasks such as arranging, formatting, removing duplicates and deleting unnecessary columns. After the data wrangling process was done, analysis 
tasks were done as well to gain more insights on the data at hand.
